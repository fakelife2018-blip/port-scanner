# Port Scanner

A Python tool that scans a target IP address for open ports and saves the results to a file.

## What it does
- Takes a target IP and port range from the user
- Scans each port and checks if it is open
- Labels common ports (HTTP, SSH, RDP, SMB, etc.)
- Saves a timestamped report to scan_results.txt

## How to run
python port_scanner.py

## Skills used
Python, socket programming, file handling, dictionaries
